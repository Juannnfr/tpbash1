# TP Bash

Script en Bash que permite gestionar archivos de alumnos mediante un menú interactivo.

## Funcionalidades

- Crear entorno de trabajo
- Procesar archivos automáticamente en background
- Listar alumnos ordenados por padrón
- Mostrar top 10 notas
- Buscar alumno por padrón
- Eliminar entorno completo (-d)

## Uso

```bash
export FILENAME=alumnos
./menu.sh
